# jameet_meta_mcp_ads

자밋(Jameet) Meta 광고 소재와 실행 플랜을 관리하는 레포입니다.

## 파일 구조

| 파일 | 내용 |
|---|---|
| `jameet_ad_creatives.md` | 광고 소재 패키지 (소재 4종 카피·이미지 + 릴스 가이드 2종 + 스토리 라인업 + 실행 플랜) |
| `agent/` | Meta Marketing API 자동 업로드 에이전트 (`agent/README.md` 참고) |

## 브랜치 규칙

- 기본 작업은 전부 `main`에서 관리합니다 (문서 중심 레포라 브랜치 분리 불필요).
- 검토가 필요한 큰 변경만 `ads/<주제>` 형식으로 브랜치를 만듭니다. 예: `ads/2026-07-신규소재`
