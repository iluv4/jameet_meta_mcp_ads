## 제품 요약 (리서치 기반)

- **자밋**: 라이프스타일 기반 자만추(자연스러운 만남 추구) 미팅·소개팅 앱
- 슬로건: "오늘 만나는 사람이 인연이 될지도 모릅니다"
- 핵심 차별점:
  1. **오늘미팅** — 당일 바로 만나는 실시간 미팅
  2. **지인 동반 미팅** — 1:1부터 친구와 함께하는 그룹 미팅
  3. **검증 시스템** — 본인인증 + 학교·직장 서류/프로필 심사 통과 회원만 가입
  4. **지인차단** — 연락처 기반으로 아는 사람 노출 차단
  5. **후불제 소개팅** — 매칭 전 대표사진만 공개, 매칭 후 전체 프로필
- 링크:
  - 웹: https://www.jameet.co.kr/
  - App Store: https://apps.apple.com/kr/app/id6743886266
  - Google Play: https://play.google.com/store/apps/details?id=jameet2.lusoft.android

| 3 | 신청 카드 도착 (사진 1장 + 짧은 메시지 + 흐릿한 프로필) | UI에 은근하게 "수락 시 서로 프로필 공개". 메시지 예: "조용한 카페 좋아해요 :)" / "퇴근하고 디저트 먹고 싶네요" / "성수 자주 가요" |

| 6 | 함께 걷는 뒷모습 | "억지 소개팅보다 이렇게 자연스럽게 만나는 게 좋더라" |

차별점: "소개팅앱에서 매칭됐다"가 아니라 **"내가 올린 주말 계획에 누군가 신청했다"** — 게시물 기반·미팅 생성·신청 시스템·사진 한 장·짧은 메시지·부담 없는 수락·실제 만남을 30초 안에 모두 노출

## 훈님께 요청할 앱 화면 리스트 (소재·릴스 제작용)

3. 미팅 지원자 내역 — 지원자 카드 (수락 전: 사진 + 짧은 메시지 + 슈퍼신청 뱃지)

5. 수락 직후 프로필 공개로 전환되는 화면
6. 매칭 완료 / 채팅 시작 화면
7. 푸시 알림 "새로운 신청자가 있습니다" (잠금화면 또는 인앱)
8. 지인차단 설정 화면 (소재 C 신뢰 앵글용)

## 자동 업로드 에이전트

`agent/` 폴더 참고 — Ads MCP 롤아웃과 무관하게 Marketing API 직접 호출로
캠페인/광고 세트/소재/광고를 일괄 생성. 사용법은 `agent/README.md`.

## Ads MCP 활성화 후 실행 플랜 (모두 PAUSED로 생성 → 검토 후 활성화)

1. **페이지 확인**: `ads_get_ad_account_pages(ad_account_id="27049951394664152")` → `page_id` 확보
2. **캠페인 생성** (CBO, 일 ₩20,000):
   ```json
   {
     "ad_account_id": "27049951394664152",
     "campaign_name": "자밋_앱설치_2026Q2",
     "objective": "OUTCOME_APP_PROMOTION",
     "buying_type": "AUCTION",
     "campaign_daily_budget": 20000,
     "special_ad_categories": "[]"
   }
   ```
   - 웹 트래픽 목적이면 `objective: "OUTCOME_TRAFFIC"` 으로 변경
3. **광고 세트 생성** (데이팅 정책상 성인 타겟 필수 — 연령 하드캡 적용):
   ```json
   {
     "ad_set_name": "자밋_KR_20-39",
     "billing_event": "IMPRESSIONS",
     "optimization_goal": "APP_INSTALLS",
     "targeting": "{\"geo_locations\":{\"countries\":[\"KR\"]},\"age_min\":20,\"age_max\":39,\"targeting_automation\":{\"advantage_audience\":0}}",
     "promoted_object": "{\"application_id\":\"<APP_ID>\",\"object_store_url\":\"https://play.google.com/store/apps/details?id=jameet2.lusoft.android\"}"
   }
   ```
4. **소재 4종 생성**: `ads_create_creative` — 위 표의 본문/헤드라인/설명/CTA + 각 소재의 이미지 URL을 `image_url`로 전달 (Meta가 서버에서 직접 가져가므로 사전 업로드 불필요)
   - 이미지는 Higgsfield 계정 라이브러리에도 보관되어 있음 (`show_generations`로 재조회 가능). 텍스트 오버레이("오늘미팅" 등)가 필요하면 디자인 툴에서 추가하거나 Higgsfield로 재생성
5. **광고 3개 생성**: `ads_create_ad` — 각 소재를 `{"creative_id": "..."}` 로 연결
